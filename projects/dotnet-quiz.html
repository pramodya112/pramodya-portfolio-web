Project Overview

A professional quiz application built with ASP.NET Core MVC to help .NET interns prepare for technical interviews. The application features 30 comprehensive questions covering C#, ASP.NET Core, OOP, LINQ, and Entity Framework. With random question selection, instant scoring, and a modern responsive UI, it provides an engaging and effective interview preparation experience with personalized results and visual feedback.

Technologies Used

Tech Stack

  • ASP.NET Core MVC: Web framework for model-view-controller architecture
  • Entity Framework Core: ORM for database operations and data management
  • SQL Server: Relational database for storing questions and results
  • C#: Primary programming language for backend logic
  • Razor Pages: View engine for dynamic HTML generation
  • HTML/CSS: Frontend styling with modern design principles
  • Session Management: ASP.NET Core sessions for user state management

Challenges & Solutions

Technical Challenges

Building a robust quiz system with randomization, scoring, and session management:

  • Random Selection: Implemented LINQ queries for efficient random question selection
  • Session Management: Used ASP.NET Core sessions to maintain user context across requests
  • Scoring Logic: Created accurate scoring algorithms with instant feedback mechanisms
  • Database Design: Structured database schema for scalable question and answer storage

Outcome & Impact

Project Results

Successfully developed a comprehensive interview preparation tool that helps .NET interns assess their technical knowledge. The application demonstrates proficiency in ASP.NET Core MVC, Entity Framework Core, and modern web development practices. The random question selection ensures varied practice sessions, while instant results provide immediate feedback for continuous improvement.
